  git credential reject

  Credentials" → find and remove the github.com entry, then retry git push.

  To generate a PAT:
  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens
  (classic)
  2. Generate with repo scope
  3. Use it as the password (or embed in the URL as shown above)

  - Routes for logout, profile, add/edit/delete expenses are placeholders
  returning placeholder strings
  - Steps 1–9 are referenced in app.py comments, suggesting a guided curriculum

  It's essentially a starter template for a Flask + SQLite CRUD app, likely used
   in a course or workshop.

❯ what is this project all about?                                               

● Spendly is a personal expense tracker web app built with Flask (Python).

  Purpose: Help users log and understand their spending — track expenses by
  category, view monthly summaries, and filter by date range.

  Tech stack:
  - Backend: Flask + SQLite
  - Frontend: HTML/CSS/JS (Jinja2 templates)
  - Testing: pytest + pytest-flask

  Planned features:
  - User registration & login/logout
  - Add, edit, delete expenses
  - Category breakdowns (Bills, Food, Health, Transport, etc.)
  - Monthly spending summaries and date filtering

  Current state: It's a teaching/workshop starter project — the UI and routes
  are scaffolded, but most functionality is left for students to implement step
  by step (Steps 1–9 referenced in app.py). The database layer (database/db.py)
  is entirely a stub.
